# Mock Dapp Prototype

## 事前準備
### [Bun](https://bun.sh/)
Bunをinstall

### [lefthook](https://github.com/Arkweid/lefthook)
homebrewでinstall
```bash
brew install lefthook 
```

## インストール
ルートディレクトリで以下のコマンドを実行
```bash
bun install
```

## ローカル開発環境を起動
```bash
bun run dev
```
